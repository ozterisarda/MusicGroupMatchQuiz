# MusicGroupMatchQuiz

A simple, entirely handwritten command-line Python quiz that matches user preferences to a music group. Developed as a foundational learning project.

This is an \*\*entirely handwritten, foundational learning project\*\* developed in Python. It serves to showcase basic programming proficiency including control flow, function usage, data handling with dictionaries, and interaction with system libraries (`webbrowser`).



The application runs a personality quiz matching the user's preferences (favorite color, animal, dessert type, and season) to one of several music groups.
